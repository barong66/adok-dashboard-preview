# Adok.ai Dashboard — preview

Статический мокап нового дашборда adok.ai по таску [asg/support#118](https://gitlab.as12as.com/asg/support/-/work_items/118).

**Что в превью:**

- Шапка «доходы» (оставлена как сейчас) — 6 карточек: today / yesterday / 7d / this month / prev month / 30d
- Realtime activity (оставлено как сейчас)
- Revenue by ad format — горизонтальные бары (Popunder / Banner / In-video / In-page push / Interstitial)
- **NEW:** Revenue by Traffic Sources — таблица `Source | Total | UV | PPV | eCPM`
- **NEW:** Revenue by Websites — таблица `Website | Total | Revenue | eCPM`

Все цифры в превью выдуманные, бэкенда нет — это только визуальный эталон для согласования дизайна перед реализацией в реальном [vue-front/src/apps/adok](https://gitlab.as12as.com/asg/vue-front/-/tree/master/src/apps/adok).

Один файл, без сборки и зависимостей.
